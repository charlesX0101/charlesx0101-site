This section describes practical labs for hands-on skill development and technical exploration. Each project below aimed to test tools, experiment with systems, or simulate real-world scenarios across networking, system support, and cybersecurity environments.

utility-nucbox

A multipurpose utility node, built using a GMK NucBoxG5 mini PC used in the homelab. It hosts core services for synchronization, DNS experiments, and secure remote access. This project includes the installation and initial base configurations of Syncthing, Pi-hole, Tailscale, and WireGuard.

c10udx0101-data_server

A rack-ready, multifunctional homelab data server running GitLab CE, Nextcloud, Samba, and PXE Boot on an enterprise level segmented VLANnetwork.

opnsense-firewall-build

A full, job-ready OPNsense firewall build for homelab and small business networks. It includes VLAN segmentation, interface rules, and secure default deny policies.

ubuntu-secure-server-setup

Secure Ubuntu Server LTS setup with SSH hardening, UFW firewall, and QEMU-based deployment.

windows-server-ad-lab

QEMU-based Windows Server 2022 lab with AD DS, user roles and system hardening via PowerShell.

unifi-wap-vlan-setup

Segmented wireless access point deployment using Ubiquiti U6+ and VLAN-based SSIDs in a production homelab network.

homelab-build

Documentation-first blueprint for setting up a home network. Includes device selection, network segmentation, diagrams, and configuration rationale.

homelab-network-validation-routine

Repeatable VLAN, DHCP, DNS, and isolation testing procedures for maintaining segmented homelab network integrity.
